# Hydroponics Knowledge Base

A complete HTML guide to DIY hydroponic systems with interactive calculators and video tutorials.

## Features

- 6 hydroponic system guides (Kratky, Dutch Bucket, DWC, NFT, Wick, Power Tower)
- Interactive nutrient calculators
- Step-by-step build instructions
- Video reference libraries
- Mobile-responsive design

## Systems Covered

### Passive Systems (No Electricity)
- **Kratky Method** - Classic passive technique for lettuce and leafy greens
- **Power Tower** - Vertical Kratky system using 5-gallon buckets
- **Wick System** - Simple capillary action system

### Active Systems (Electricity Required)
- **DWC (Deep Water Culture)** - Highly oxygenated system for fast growth
- **NFT (Nutrient Film Technique)** - Commercial-grade continuous flow system
- **Dutch Bucket** - Bato bucket setup for tomatoes and peppers

## Usage

1. Open `index.html` in any web browser
2. Select your preferred hydroponic system
3. Follow the interactive guide with calculators and videos

All files are self-contained HTML pages - just download and use locally or deploy to any static hosting.

## Target Audience

Perfect for home gardeners and DIY enthusiasts wanting to learn hydroponics from the ground up.
